How registration works

  1. 1Your registration is submitted as a GitHub / Codeberg Issue with label fdp-registration.
  2. 2A maintainer reviews it (or it's auto-approved if the catalog URL validates).
  3. 3A YAML file is added to registered-fdps/ and the pipeline runs.
  4. 4Your FDP appears in index.ttl and the browse UI within 24 h.
